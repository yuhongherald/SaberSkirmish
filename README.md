# Saber Skirmish

This repository contains sample code used for the development of our game. Enjoy!

[![Watch the video](docs/images/SaberSkirmish.gif)](https://www.youtube.com/watch?v=YUS0_jPUu_Q)

## About

Saber Skirmish is an arcade-style rail-shooter genre game, made in Unity. The project was developed as part of a summer programme at NUS (CS3241R) to practice Computer Graphics and animation concepts. Contributors for the project are Yu Hong Herald and Yu Pei Henry.
